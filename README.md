## 변수
* 프로그래밍의 첫번째 작업은 필요한 데이터들을 반드시 **변수(저장소)**로 저장해야 한다.
## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box;` <!-- 박스를 생성하겠다. -->
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box=1;`<!-- 박스라는 변수를 생성해서 1을 대입하겠다. 오른쪽에서 왼쪽으로 -->
* `생성변수명 대입연산자 대입값;`
    * `box=1;` <!-- 빈 박스를 준비하겠다. -->
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합 (숫자는 영문 뒤로)
## 변수 작성시 주의 사항
* 변수의 이름은 문자(a~z,A~Z), 숫자(0~9),달러($) 밑줄(_)이 포함됩니다.
* 변수의 이름은 숫자로 시작할 수 없습니다.
* 변수의 이름은 대시(-)나 마침표(,) 기호를 사용할 수 없습니다.
키워드, 예약어 종류 -> delete, this, false, true, if, default, var, new, class 등
* 모든 변수는 대/소문자를 구별합니다. score와 Score는 다른 변수입니다.
* 변수가 저장할 정보의 종류를 표현하는 의미하는 단어를 사용해야 합니다!
## 변수 표기법 종류
* 캐멀 표기법
    newName
    wedsitecode
* 파스칼 표기법
    NewName
    Wedsitecode
* 밑줄 표기법
    new_Name

## 객체, 속성, 메서드 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 객체를 메서드 방법으로 실행함
* `객체.속성.메서드();` 
* `객체.속성.속성.메서드();` 
* `객체.메서드().속성;` 
* `객체.메서드().메서드();` 
* `객체.메서드(메서드());` 
## 데이터타입 구분 함수 `typeof()`
* css의 배경 색상으로 영역 구분하는 것처럼 데이터 타입 구분 목적 사용
1. 특정 데이터 타입을 담아둔 변수를 생성
* `let num = 1;`
* `let str = '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수 확인
* `console.log(typeof(num));` //number(숫자)
* `console.log(typeof(str));` // string(문자)
## 피연산자가 숫자인 경우 -> 산술 연산자
* a + b 더하기) 두개의 피연산자를 더한다
* a - b 빼기) 왼쪽 피연산자에서 오른쪽 피연산자를 뺀다.
* a * b 곱하기) 두개의 피 연산자를 곱한다.
* a / b 나누기) 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 몫이다.
* a % b 나머지) 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지이다.
* a ** b 거듭제곱) 왼쪽의 피연산자의 오른쪽 피연산자의 값만큼 곱한 값이다.
계산할 수 없는 경우는 NaN 처리 ex)'two' * 2 = NaN
## 함수 종류
* `prompt('질문')` : 질문답변 창 띄우는 함수
    * `input` 태그로 입력 받는 사용자 입력값도 모두 문자로 처리
    * 쇼핑몰의 수량 `input`도 문자로 기본적으로 인식됨.
* `Number(숫자로 변환하고 싶은 데이터 또는 함수)` : 질문답변 창 띄우는 함수
* `prompt(Number())` : **X** prompt 질문 입력 위치에 다른 함수 작성
* `Number(prompt())` : **O** Number 괄호안은 변환대상 prompt 함수 작성 가능
## 복합 대입 연산자
* 연산자 : `+=`
    * 예시 : `a+=b`
    * 뜻 : `a=a+b`
    * 예시풀이 : `기존 a 값에 b를 더하여 a에 대입한다.`
* 연산자 : `-=`
    * 예시 : `a-=b`
    * 뜻 : `a=a-b`
    * 예시풀이 : `기존 a 값에 b를 빼서 a에 대입한다.`
* 연산자 : `*=`
    * 예시 : `a*=b`
    * 뜻 : `a=a*b`
    * 예시풀이 : `기존 a 값에 b를 곱해서 a에 대입한다.`
* 연산자 : `/=`
    * 예시 : `a/=b`
    * 뜻 : `a=a/b`
    * 예시풀이 : `기존 a 값에 b를 나눠서 a에 대입한다.`
* 연산자 : `%=`
    * 예시 : `a%=b`
    * 뜻 : `a=a%b`
    * 예시풀이 : `기존 a 값에 b를 나눈 나머지값을 a에 대입한다.`

## 함수의 종류
* `function`-> 함수 선언시 함수라는 것을 나타내는 키워드
* `함수이름` -> 개발자가 만든 함수의 의미있는 이름을 작성한다
* `전달인자` -> 매개변수, 인수 등 여러가지 용어를 사용한다. 함수 내에 데이터를 전달하는 변수명이라 보면 되며 전달 데이터가 많을 때 객체 타입으로 전달하기로 한다.
* `중괄호{}` -> 함수 영역의 시작과 끝을 나타내며 함수의 반복적인 실행 내용을 작성한다.
